# OneFit

## Justificativa do Tema

O tema de academia foi escolhido por ser um contexto comum e relevante no dia a dia das pessoas. Sistemas voltados para academias permitem aplicar diversos conceitos de desenvolvimento de software, como cadastro de usuários, gerenciamento de treinos, controle de planos e organização de dados.

Além disso, esse tipo de aplicação possui regras de negócio claras, o que facilita a modelagem do sistema e o entendimento das funcionalidades. Isso torna o projeto adequado para praticar conceitos como relacionamento entre entidades, persistência de dados e construção de APIs.
